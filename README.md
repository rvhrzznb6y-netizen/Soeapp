# SOE Software Store — تطبيق PWA

## رفع على GitHub Pages (3 خطوات)

### 1. ارفع على GitHub
- New repository باسم: soe-app
- ارفع جميع الملفات من هذا ZIP

### 2. فعّل GitHub Pages  
Settings → Pages → Branch: main → Save

### 3. افتح على الآيفون
- افتح Safari على الآيفون
- ادخل: https://username.github.io/soe-app
- اضغط Share → Add to Home Screen
- ✅ يظهر التطبيق على شاشتك كأيقونة!

---

## الملفات
- index.html — الموقع كاملاً
- manifest.json — بيانات التطبيق
- sw.js — Service Worker (يعمل بدون إنترنت)
- icon-192.png — أيقونة التطبيق
- icon-512.png — أيقونة عالية الدقة
- icon-180.png — Apple Touch Icon
